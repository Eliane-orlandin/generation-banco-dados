# Generation Banco de Dados

Este repositório é destinado a guardar scripts SQL relacionados aos estudos de banco de dados da Generation Brasil, do curso JAVA 84.

## Objetivo

- Organizar scripts SQL desenvolvidos nas atividades e estudos do curso.
- Manter um histórico de consultas, criação de tabelas e manipulação de dados.

## Estrutura

- `sql/` — diretório para armazenar todos os arquivos e scripts SQL.

## Sobre

Este repositório serve como referência prática para aprender conceitos de banco de dados e aplicar comandos SQL nas aulas e exercícios do curso JAVA 84 da Generation Brasil.
